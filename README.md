## Overview

Mentora AI is an intelligent learning platform that transforms static PDFs into dynamic, interactive quizzes. Built for students who want to learn faster and teachers who need to create engaging content effortlessly, Mentora AI leverages artificial intelligence to automate quiz generation, provide instant feedback, and make studying smarter — not harder.

## Problem

Traditional studying is inefficient. Students spend hours reading through dense PDFs without a clear way to test their understanding. Teachers, on the other hand, face the repetitive burden of manually crafting quizzes, homework, and revision materials. Static content is boring, and without active recall, retention suffers. The education gap isn't about a lack of content — it's about a lack of smart tools.

## Solution

Mentora AI bridges this gap by using AI to turn any uploaded PDF into a fully structured quiz within seconds. Students get instant feedback with correct answers, and teachers save countless hours of manual content creation. The result is a faster, more engaging learning experience that works for everyone — in the classroom, at home, or anywhere in between.

## Key Features

- **AI Quiz Generation** — Upload a PDF and get a complete quiz generated automatically.
- **Instant Feedback & Answers** — Know right away whether you got it right, with correct answers displayed.
- **Smart Learning Experience** — Active recall made simple; learn by doing, not just reading.
- **Clean & Modern UI** — Beautiful, intuitive interface designed for focus and usability.
- **Fast & Scalable** — Built on Next.js for lightning-fast performance and seamless scalability.

## Tech Stack

- **Next.js** — React framework for production-grade SSR and routing
- **AI APIs** — Powered by large language models for intelligent content understanding
- **Tailwind CSS** — Utility-first styling for a polished, responsive design
- **TypeScript** — Type-safe code for reliability and maintainability
- **Modern Frontend Architecture** — Server components, API routes, and edge-ready deployment
